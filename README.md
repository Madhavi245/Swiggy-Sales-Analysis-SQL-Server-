# Swiggy-Sales-Analysis-SQL-Server-
Swiggy Sales Analysis (SQL Server) – A data analysis project using SSMS to clean and transform Swiggy food delivery data, detect duplicates, and build a Star Schema (dim_date, dim_location, dim_restaurant, dim_category, dim_dish, fact_orders). SQL queries generate KPIs such as total orders, revenue, ratings, and food performance insights. 📊🍽️
